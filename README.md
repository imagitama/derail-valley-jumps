# Derail Valley Jumps

A mod for the game [Derail Valley](https://store.steampowered.com/app/588030/Derail_Valley/) that lets you jump your train!

## Bindings

Open the mod settings (Ctrl+F10) to view and change bindings.

By default "space" for jump and numpad numbers for roll/spin/turn.

## Install

Download the zip and use Unity Mod Manager to install it.

## Development

Template from https://github.com/derail-valley-modding/template-umm

Created in VSCode (with C# and C# Dev Kit extensions) and MSBuild.

1. Run `msbuild` in root to build

## Publishing

1. Run `.\package.ps1`
